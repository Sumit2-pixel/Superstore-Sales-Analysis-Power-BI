# 📊 Superstore Sales Analysis (Power BI)

## 📌 Overview

This is my first Power BI project where I analyzed sales and profit data using the Superstore dataset.

## 🎯 Objectives

* Analyze sales across regions
* Identify profitable categories
* Track performance trends

## 📊 KPIs

* Total Sales
* Total Profit
* Total Orders

## 📈 Key Insights

* Technology category generated highest profit
* West region has highest sales
* Some categories show negative profit

## 🛠 Tools Used

* Power BI
